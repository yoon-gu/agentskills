Agent Skills는 지침, 스크립트, 리소스로 구성된 폴더이며, 에이전트가 이를 탐색하고 활용해 작업을 더 정확하고 효율적으로 수행할 수 있게 합니다.

## 왜 Agent Skills인가요?

에이전트의 능력은 빠르게 향상되고 있지만, 실제 업무를 안정적으로 수행하는 데 필요한 컨텍스트가 부족한 경우가 많습니다. 스킬은 에이전트가 필요할 때 절차적 지식과 회사, 팀, 사용자 특화 컨텍스트를 불러올 수 있도록 하여 이 문제를 해결합니다. 스킬 세트에 접근할 수 있는 에이전트는 수행 중인 작업에 맞춰 역량을 확장할 수 있습니다.

**스킬 작성자**: 기능을 한 번 구축하면 여러 에이전트 제품에 배포할 수 있습니다.

**호환 에이전트**: 스킬 지원을 통해 최종 사용자가 별도 설정 없이 에이전트에 새로운 기능을 부여할 수 있습니다.

**팀과 엔터프라이즈**: 조직 지식을 이식 가능하고 버전 관리되는 패키지로 축적할 수 있습니다.

## Agent Skills로 무엇을 할 수 있나요?

- **도메인 전문성**: 법률 검토 프로세스부터 데이터 분석 파이프라인까지, 전문 지식을 재사용 가능한 지침으로 패키징합니다.
- **새로운 기능**: 에이전트에 새로운 기능을 부여합니다(예: 프레젠테이션 생성, MCP 서버 구축, 데이터셋 분석).
- **반복 가능한 워크플로**: 여러 단계의 작업을 일관되고 감사 가능한 워크플로로 전환합니다.
- **상호운용성**: 스킬 호환 에이전트 제품 간에 동일한 스킬을 재사용합니다.

## 도입 현황

Agent Skills는 주요 AI 개발 도구에서 지원됩니다.

## 오픈 개발

Agent Skills 형식은 [Anthropic](https://www.anthropic.com/)에서 처음 개발되었고, 이후 개방형 표준으로 공개되어 점점 더 많은 에이전트 제품에서 채택되고 있습니다. 이 표준은 더 넓은 생태계의 기여를 환영합니다.
