# Agent Skills

[Agent Skills](https://agentskills.io)는 에이전트에 새로운 역량과 전문성을 부여하기 위한 단순하고 개방적인 형식입니다.

스킬은 지침, 스크립트, 리소스로 구성된 폴더이며, 에이전트가 이를 탐색하고 활용해 특정 작업을 더 잘 수행할 수 있습니다. 한 번 작성해 여러 곳에서 사용할 수 있습니다.

## 시작하기

- [문서](https://agentskills.io) - 가이드와 튜토리얼
- [명세](https://agentskills.io/specification) - 형식 세부 사항
- [예제 스킬](https://github.com/anthropics/skills) - 가능한 활용 사례 보기

이 저장소에는 명세, 문서, 레퍼런스 SDK가 포함되어 있습니다. 예제 스킬 목록은 [여기](https://github.com/anthropics/skills)에서 확인할 수 있습니다.

## 소개

Agent Skills는 [Anthropic](https://anthropic.com)이 유지 관리하는 개방형 형식이며, 커뮤니티 기여를 환영합니다.
